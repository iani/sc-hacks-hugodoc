Operator Overview

sc-hacks uses binary operators to shorten and help classify the coding of common operations such as playing a synth or a pattern. The syntax of a binary operator in SuperCollider is: receiver operator[optional: .adverb] argument For example: // receiver operator adverb argument { WhiteNoise.ar(0.1) } +>.newenvir \mysynth Here is a list of operators introduced in sc-hacks. List of Operators Planned are 4 operators taking functions as argument, 4 operators taking event as argument, and 5 operators taking a symbol as argument. »
