HUGO docdock - Easy diagrams

Some demos

The following diagrams are produced by the mermaid tool, just by some simple code as shown below. No plugins required. Use docdock theme with hugo.

Demo 1 - simple flow chart

Unfold to see code

Code:

graph LR;
    A[Hard edge] -->|Link text| B(Round edge)
    B --> C{Decision}
    C -->|One| D[Result one]
    C -->|Two| E[Result two]

graph LR; A[Hard edge] -->|Link text| B(Round edge) B --> C{Decision} C -->|One| D[Result one] C -->|Two| E[Result two]
